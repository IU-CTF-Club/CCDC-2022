# Edit files via shared TMUX sessions

## Open a file

```
$ ./topen.sh FILENAME
```

This command will start a tmux server if not already running.

## List all open sessions

```
$ ./tlist.sh
```

## Attach to an open session

[fzf](https://github.com/junegunn/fzf) is recommended.

```
$ ./tattach.sh
```
